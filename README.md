TradeMatrix – Stock Trading Simulation System
TradeMatrix is a Java-based stock trading simulator featuring an interactive GUI, real-time order book updates, and a functional matching engine. Users can place buy/sell orders, track market activity, and view dynamic price/volume tables through a clean and user-friendly interface.


Features
Interactive GUI built with Java Swing/JavaFX
Buy/Sell order submission with validation
Real-time order book showing price, volume, and user
Automated matching engine that simulates stock exchange behavior
Live event log that records trades and system activity
Color-coded interface for quick visual feedback


Tech Stack
Java (OOP, event-driven architecture)
Swing / JavaFX for GUI
Custom data structures for order handling
IDE: IntelliJ / VS Code / Eclipse
